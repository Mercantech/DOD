# DOD
Deploy or Die - Fra kode til Internet. Deployment, Sikkerhed og DevOps


### Fag og målpinde under DoD


#### 20738 - DevNet

1. Lærlingen kan benytte basis Python programmeringscript
2. Lærlingen kan benytte basis Linux shell-kommandoer
3. Lærlingen kan implementere og udvikle simple DevNet-miljøer (API-kald).
4. Lærlingen kan bruge gældende standarder til at udvikle DevNet-miljøer.
5. Lærlingen kan oprette API-forespørgsler via sikre protokoller f.eks. HTTPs.
6. Lærlingen kan forklare grundlæggende netværksbegreber.
7. Lærlingen kan benytte værktøjer til at deploye og sikre data i et cloudmiljø

[EUD Enkeltfag - 20738](https://www.eud.uddannelsesadministration.dk/Soeg/EUDEnkeltfag/Detaljer.aspx?FAG_ID=225258)

#### 22253 - Informationssikkerhed og awareness

1. Lærlingen kan redegøre for forskellige trusler som fx malware, phishing, ransomware.
2. Lærlingen kan redegøre for de grundlæggende principper i NIS2 og CRA's formål og anvendelse i virksomheden.
3. Lærlingen kan redegøre for etiske principper i arbejdet med data- og it-sikkerhed.
4. Lærlingen kan redegøre for awareness tiltag i virksomheden fx adgangskoder, tofaktorautentificering, og sikkerhedsopdateringer.
5. Lærlingen kan på et grundlæggende niveau indtænke risikovurdering ind i løsningen af konkrete hverdagssituationer.

[EUD Enkeltfag - 22253](https://www.eud.uddannelsesadministration.dk/Soeg/EUDEnkeltfag/Detaljer.aspx?FAG_ID=228980)
